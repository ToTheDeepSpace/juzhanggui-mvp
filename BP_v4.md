## 一、产品矩阵

### 剧掌柜（店家端）
演绎剧场/密室/沉浸式戏剧的排期 + 抢车 + 签到 + 管理工具。

**线上**：https://script-scheduler.vercel.app

### 灵契（创作者端）
卡司、Coser、摄影师、妆造师的个人主页 + 档期展示 + 粉丝联系平台。

**线上**：https://lingqi.vercel.app

## 二、当前功能

### 剧掌柜核心功能

| 功能 | 说明 |
|------|------|
| **排程管理** | 抢车模式，满员自动发车 |
| **生命周期** | 待锁车→锁车(定金)→排班(配卡司)→开始→结束登记 |
| **冲突检测** | 选房间/卡司时自动高亮冲突 |
| **玩家签到** | 二维码扫码，验证码登录，角色+性别选择 |
| **客服代填** | 同一页面连续添加多位玩家 |
| **结束登记** | 正常/炸车/流车/其他 + 打扫确认 + 评价二维码 |
| **矛盾登记** | 内嵌在结束流程中，自动同步 |
| **卡司管理** | 关联灵契主页，一键跳转 |
| **数据看板** | 进行中/已结束切换 |

### 灵契核心功能

| 功能 | 说明 |
|------|------|
| **创作者主页** | 个人资料、作品集、社交链接 |
| **可接服务** | 明码标价，自由设置 |
| **档期日历** | 标记可约日期 |
| **粉丝联系** | 申请联系方式，创作者审核通过 |
| **入驻免费** | 手机号+密码注册 |

## 四、竞品定位

| 竞品 | 核心 | 与我们的区别 |
|------|------|-------------|
| 谜圈 | 组局+评价（流量） | 我们是运营工具+创作者经济，互补不冲突 |
| 千岛 | 潮玩交易（剧本杀是频道之一） | 不做店务，不做创作者经济 |
| 剧帮主 | 店务工具（功能全但无亮点） | 我们有抢车模式+灵契创作者生态 |
| 巨量星图 | 品牌→达人→线上广告 | 我们是粉丝→创作者→线下体验，不同赛道 |
