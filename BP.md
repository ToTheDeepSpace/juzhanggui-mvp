## 一、这是一件什么事

做一个覆盖剧本杀行业全产业链的数字化运营平台。

把发行商、店家、卡司/DM、玩家四方连在一起——
让发行商看到真实的市场数据，让店家管得轻松，
让卡司有尊严有成长，让玩家拼本更方便。

> 剧掌柜 = 剧本杀的"猫眼"。猫眼做的是电影行业的基础设施（排片、售票、数据），我们做剧本杀行业的基础设施。

### 为什么现在是个机会

**行业还在扩张，但工具很落后**

- 大量门店还在用 Excel + 微信群管排期、管卡司、管物资
- 卡司/DM 普遍被当"耗材"用，高离职率是行业痛点（[需验证] DM 年离职率约 40-60%）
- 没有一家平台同时拥有发行商、店家、卡司、玩家的完整数据

**技术条件已经成熟**

- 中医 SaaS、餐饮 SaaS 已经跑通了多租户 + 数据隔离的模式
- Next.js + PostgreSQL 可以快速做出可演示的产品
- AI 辅助推荐已经可以在不增加人力的情况下做个性化服务

**你在正确的时间点**

- 行业经历了 2018-2022 的野蛮生长，2023-2024 开始洗牌——淘汰的是差店，留下的是愿意花钱提升效率的好店
- 这个时间点进来的 SaaS 产品，面对的是"已经活下来的、有付费意愿的"客户

## 三、2026 年行业新趋势

### 城限情感本当道

2024-2026 年间，剧本杀内容生态发生了结构性变化：

- **城限本成为竞争核心**：优质城限本售价数千元起，独家本更是有钱也买不到。店家之间的竞争从"谁能拼到人"变成了"谁能抢到好本"
- **情感本主导市场**：情感本已经成为最主流的品类，玩家不再只满足于推理，更追求沉浸式的情感体验
- **恋陪卡司（恋爱陪伴型 DM）爆发**：2025 年起，恋陪本成为行业最大热点——通过 NPC 角色深度演绎和陪伴，提供"情感代餐"体验。头部恋陪 DM 定价可达 2000 元/8 小时

> 这些趋势意味着：店家面临的运营复杂度急剧上升——管排期、管 DM、管本、管客户的难度都在翻倍。这正是 SaaS 工具的切入时机。

## 四、竞品全景图：谜圈、千岛、米花星球

剧本杀数字化赛道已有三个主要玩家。以下是基于公开信息的完整对比。

### 竞品一：谜圈 — 行业老大，QR 码飞轮

谜圈是目前剧本杀行业最大的数字化平台。

| 维度 | 数据 |
|------|------|
| **公司** | 深圳市你好时代网络有限公司 |
| **创始人** | 别赛男（连续创业者，10 年经验） |
| **成立** | 2020 年 8 月，2021 年 7 月上线 |
| **融资** | 自筹 2000 万 + 天使轮 1000 万 = 共约 3000 万 RMB |
| **入驻店家** | **24,000+ 家**（2023.4 数据） |
| **发行商** | 1,000+ 家 |
| **剧本覆盖** | 9,000+ 个 |
| **下载量** | 4,000,000+ |
| **盈利状态** | **2023 年 2 月已实现盈亏平衡** |

**核心产品：**
- 剧本评分 + 测评（类似大众点评）
- 组局拼本（同城匹配队友）
- 店家发现 + 门店加盟
- 谜圈商家版（店家主端）
- 密室逃脱支持

**杀手锏：QR 码飞轮**
谜圈最可怕的地方在于——**每个剧本的最后一页都印着他们的二维码**。每个打完本的玩家自然看到，扫码就能看测评、组新局。这是零成本的病毒式增长飞轮：剧本越多 → QR 码越多 → 用户越多 → 店家/发行商越离不开。

**商业模式：** "剧本杀的猫眼"——点评 + 售票 + 发行。合作疯狂剧制做票房分成。

### 竞品二：千岛剧本杀 — 背靠巨头的兴趣社区

千岛不是一家独立公司，是一个巨型多垂类兴趣平台下的剧本杀频道。

| 维度 | 数据 |
|------|------|
| **母公司** | 回响科技（原潮玩族） |
| **融资** | **B 轮近 1 亿美元**（红杉中国 + 高榕资本领投，2021.12） |
| **总用户** | **约 3000 万**（2026.3 数据） |
| **年 GMV** | 约 15 亿元 |
| **核心业务** | 潮玩手办交易（泡泡玛特二级市场） |

**千岛的核心不是剧本杀。** 它是一个潮玩交易平台，剧本杀、卡牌、谷圈（动漫周边）、三坑（汉服/Lolita/JK）都是后来扩展的兴趣垂类。千岛剧本杀只是其中一个频道。

千岛剧本杀的功能：剧本拼车、评分、店家入驻、微信小程序、商家端、优惠券（19.9 代 50）。

**弱点：** 剧本杀不是千岛的主营业务。它的资源（技术、运营、BD）优先服务于潮玩交易——那才是 15 亿 GMV 的现金牛。

## 剧掌柜的差异化：不做流量平台，做运营系统

```
谜圈 = 剧本杀的大众点评 + 猫眼（评价 + 售票）
千岛 = 潮玩界的闲鱼（交易 + 社区，剧本杀是附带的）
米花星球 = 剧本杀的 Roblox（UGC 引擎）

剧掌柜 = 店家的操作系统 + DM 的成长平台
```

**为什么这条路有机会：**

1. **谜圈验证了市场**：24000 家店 + 盈亏平衡 = 这个赛道真实存在且能赚钱
2. **但谜圈没解决运营问题**：店家最痛的不是"找不到客人"，是"管不好店"
3. **城限本 + 恋陪卡司让运营复杂度翻倍**：2026 年的店家比 2023 年更需要专业工具
4. **DM 成长体系 = 护城河**：一旦卡司在这套体系里建立了职业身份，跳槽成本极高
5. **三家都在抢流量入口，我们在抢运营入口**——流量可以买，运营习惯一旦养成很难迁移

## 四、我们解决什么问题

### 店家的痛苦

| 问题 | 现状 | 我们的方案 |
|------|------|-----------|
| 排期靠脑子 | 排一个本要翻半天表 | 日历拖拽排期，冲突自动检测 |
| 卡司被用废 | 连轴转没人管 | 卡司疲劳预警 + 工作量统计 |
| 道具管理混乱 | 丢了不知道谁用的 | 按排期记录领用、盘点、损耗 |
| 成本算不清 | 不知道一场本赚了多少 | 实时成本核算（工资+道具+房租分摊） |
| 客人靠微信拼本 | 效率低、体验差 | 客人自主查看排期、选角色、扣费报名 |

### 卡司/DM 的痛苦

| 问题 | 现状 | 我们的方案 |
|------|------|-----------|
| 没有晋升通道 | 干几年就转行 | DM 成长体系（学徒→金牌），等级对应收入 |
| 不知道排班 | 每天等通知 | 个人排班日历，随时查看 |
| 不知道收入 | 算不清自己拿了多少 | 薪资明细（基础+提成），每月可查 |
| 想休假不知道找谁 | 微信发消息不一定批 | 申请休假流程，透明审批 |
| 有本事不知道怎么变现 | 只能靠口碑 | 改本版税系统，作品上传→店家付费使用 |

### 发行商的痛苦

| 问题 | 现状 | 我们的方案 |
|------|------|-----------|
| 不知道本卖得怎么样 | 全靠店家反馈 | 实时销售数据，哪个城市/哪个店/哪个本 |
| 不知道玩家真实反馈 | 全靠豆瓣/小红书 | 评价数据从"文笔/流程/氛围"三维度拆解 |
| 不知道本被改成什么样 | 完全不知道 | 店家改本记录汇总，知道哪些本被改得最多 |
| 新本不知道推给谁 | 群发广告 | 精准推送给相关店家（根据历史偏好） |

## 五、核心壁垒

> 这个系统最难被复制的部分是什么？

### 第一层：DM 成长体系认证

行业第一套 DM 职业化标准。

```
学徒（跟训考证）→ 初级DM（能带本）
  → 正式DM（优先排班）→ 高级DM（高提成）
    → 金牌DM（可培训新人）
```

当 DM 习惯了这套成长体系，**他们就会留在这个平台**——就像程序员留在 GitHub 而不是自建 Git 服务器。

### 第二层：全产业链数据

同时拥有发行商数据（哪些本卖得好）、店家数据（哪个店用得最多）、卡司数据（哪个 DM 带得好）、玩家数据（哪个本受欢迎）——

**这个数据护城河一旦建立，没有人能绕开你。**

### 第三层：卡司改本生态

让有能力的卡司把自己的改本记录上传平台，店家付费使用。这不只是增收工具，这是**让整个行业的内容创作者都留在你这里**。

## 六、单位经济模型（Unit Economics）

### 单店经济

| 项目 | 基础版（299/月） | 专业版（599/月） | 旗舰版（999/月） |
|------|-----------------|-----------------|-----------------|
| 月 ARPU | 299 元 | 599 元 | 999 元 |
| 年 ARPU | 3,588 元 | 7,188 元 | 11,988 元 |
| 预期占比 | 60% | 30% | 10% |
| 加权年 ARPU | — | — | **约 5,500 元** |

### 获客成本（CAC）

| 渠道 | 单店获客成本 | 备注 |
|------|-------------|------|
| 人脉推荐（前 100 家） | ~200 元 | 熟人介绍+请吃饭 |
| 地推扫街（100-500 家） | ~800 元 | 人力+交通+物料 |
| 内容营销（500+ 家） | ~500 元 | 短视频+行业社群 |
| 发行商推荐 | ~300 元 | 分佣模式 |

**加权 CAC：约 500 元/店**

### 回本周期（Payback Period）

```
CAC ÷ 月 ARPU = 500 ÷ 550 ≈ 1 个月
```

即 **1 个月内回本**。这在 SaaS 行业属于极优秀的水平。

### LTV（客户生命周期价值）

假设月流失率 3%（即年留存率 ~70%，保守估计）：

```
LTV = 月 ARPU ÷ 月流失率 = 550 ÷ 3% ≈ 18,333 元
LTV/CAC = 18,333 ÷ 500 ≈ 37x
```

### 收入里程碑预估

| 阶段 | 付费门店数 | 月经常性收入（MRR） | 年化收入（ARR） |
|------|-----------|---------------------|----------------|
| 第 6 个月 | 50 家 | ~2.75 万 | ~33 万 |
| 第 12 个月 | 200 家 | ~11 万 | ~132 万 |
| 第 24 个月 | 500 家 | ~27.5 万 | ~330 万 |
| 第 36 个月 | 2000 家 | ~110 万 | ~1320 万 |

> 以上仅计算 SaaS 订阅收入，不含发行商数据服务、改本平台抽成、周边分销等增量收入。

## 八、推广策略

### 第一步：单点验证（0 → 10 家店，0-3 个月）

- 找 5-10 家认识的店，免费用三个月
- 目标：验证产品有没有人用、功能有没有解决痛点
- 核心指标：周活跃率 > 60%，NPS > 30
- 同步做 DM 成长体系的内测

### 第二步：口碑扩散（10 → 100 家店，3-9 个月）

- 做出 DM 认证体系，让卡司主动帮你推广
- 在剧本杀内容账号（抖音/B站）投放 demo 视频
- 找行业大V联合开发
- 核心指标：月留存率 > 80%，NPS > 40

### 第三步：规模化（100 → 500 家店，9-18 个月）

- 建立地推团队，扫当地剧本杀店
- 发行商联动（发行商推荐给他们的合作店家）
- 核心指标：月流失率 < 5%，MRR > 10 万

## 十、为什么是你来做

### 三重稀缺性

1. **行业理解 + 内容能力**：深度体验 25 个本、35 场游戏，写剧本杀内容——你懂玩家的真实痛点，也懂店家的运营逻辑。行业里的"玩家"很多，但"能写内容的深度玩家"很少。

2. **技术落地能力**：自己能写代码、能跟 AI 协作开发。这意味着 MVP 阶段不需要烧钱招技术团队——你自己就能把产品做出来。这是绝大多数行业创业者没有的优势。

3. **AI 原生思维**：你不是在用 AI 辅助开发，你是从第一天就用 AI Agent（Hermes）作为协作伙伴。这让你一个人的产出接近一个小团队。
