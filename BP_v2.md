> **版本**：BP v2.0（重构版）

## 一、一句话

**剧掌柜 = 沉浸式娱乐行业的「秀动 × 抖音粉丝经济」。**

让卡司成为明星，让粉丝为热爱付费，让店家轻松运营。

---

## 二、我们发现了一个结构性变化

### 2.1 卡司不再是耗材，是资产

三年前，剧本杀店的 DM 是可替换的耗材。来了干、干累走、走了再招。

**但现在不是了。**

| 变化 | 原因 | 结果 |
|------|------|------|
| 恋陪卡司爆发 | 2025 年起恋陪本成为行业最大热点 | 头部恋陪 DM 单场收入可达 2000 元 |
| 卡司粉丝化 | 玩家追特定 DM 带本 | 好卡司自带流量，店家抢着签 |
| 打榜经济出现 | 幽州剧场打榜最高 24 万元/人 | 粉丝愿意为好卡司付费 |

**结论：卡司 = 沉浸式娱乐行业的 KOL，不是螺丝钉。**

### 2.2 沉浸式娱乐正在爆发

沉浸式娱乐 ≠ 剧本杀。

这是一个正在快速扩张的赛道：

```
剧本杀（你已验证）
├── 演绎剧场（客单价 ¥250+，月流水 10-30 万）
├── 密室逃脱（客单价 ¥150-300，翻台快，2 小时/场）
├── 沉浸式戏剧（客单价 ¥500+）
├── LARP / 真人角色扮演（新兴）
├── VR 体验馆（快速增长）
└── IP 主题店 / 三坑体验店
```

**共同痛点：** 排期管理 + 演员/工作人员管理 + 粉丝运营 + 会员预约。

只要有人演、有人看、需要排期——剧掌柜就能用。

### 2.3 市场规模重新估算

| 层级 | 含义 | 规模 |
|------|------|------|
| **TAM** | 中国沉浸式娱乐运营 SaaS + 粉丝经济市场 | ~1000 亿元/年 |
| **SAM** | 沉浸式剧场门店（含密室、LARP 等）| ~5 万家店，约 50 亿元/年 SaaS 市场 |
| **TAM 粉丝经济** | 卡司打榜、委托、周边 | ~100 亿元/年（参考直播打赏行业渗透率）|
| **SOM** | 初期触达的 500 家演绎剧场 | ~2500 万元/年 SaaS + 粉丝经济抽成 |

## 三、产品架构

### 3.1 三边市场模型

```
      卡司（创作者）
      ↙      ↘
  玩家（粉丝）←→ 店家（场地）
```

**卡司为什么来：** 免费个人主页 → 经营粉丝 → 接委托/打榜 → 变现  
**玩家为什么来：** 追喜欢的卡司 → 看排班/预约 → 打榜支持 → 粉丝团  
**店家为什么来：** 好卡司都在这里 → 玩家也在这里 → 顺带用排期管理

### 3.2 功能模块

#### 卡司端（独立个人主页）

| 功能 | 说明 | 变现方式 |
|------|------|---------|
| 个人主页 | 照片、风格标签、作品集、评分 | 免费（吸引入驻） |
| 排班日历 | 在哪些店、什么时间有场次（跨店） | 免费 |
| 粉丝数据 | 粉丝数、打榜排名、热度曲线 | 免费 |
| 可接委托 | 生日祝福、专属场、教学、陪玩 | 平台抽成 10-20% |
| 粉丝团 | 月卡订阅制，类似 B 站大航海 | 平台抽成 15% |
| 社交链接 | 抖音/小红书/微博引流 | 免费 |
| 成长体系 | 等级、成就、认证 | 免费（增强粘性） |

#### 玩家端

| 功能 | 说明 |
|------|------|
| 关注卡司 | 订阅卡司动态 |
| 卡司排班日历 | 查看喜欢的卡司在哪家店、什么时候有场次 |
| 打榜/送礼 | 给喜欢的卡司打榜（¥1-99999） |
| 委托预约 | 约卡司的专属场次 |
| 粉丝团订阅 | 月卡支持卡司 |
| 历史场次 | 自己参与过的所有演出 |
| 签到评价 | 到场签到 + 评分 |

#### 店家端

| 套餐 | 价格 | 功能 |
|------|------|------|
| **基础版** | **免费** | 排期管理 + 卡司管理 + 房间管理 |
| **专业版** | **¥599/月** | 基础版 + 会员管理 + 数据报表 + 卡司热度数据 |
| **旗舰版** | **¥1999/月** | 专业版 + 成本核算 + 连锁店管理 + 专属客服 |

**平台收入：**
- SaaS 订阅费（店家付）
- 打榜抽成 15%（玩家付 → 卡司 → 平台抽成）
- 委托抽成 15%（玩家付 → 卡司 → 平台抽成）
- 粉丝团抽成 15%（玩家月费 → 平台抽成）

## 四、竞争格局 & 差异化

### 4.1 赛道玩家全景

```
流量平台（组局/评价）        运营工具（店务）
       │                        │
 谜圈 ╯                        ╰ 剧帮主
 千岛 ╯                        ╰ 剧本杀伴侣
 米花星球 ╯                    ╰ 纳客/云上铺（通用SaaS套壳）
       │                        │
 谁都没做：卡司粉丝经济       谁都没做：创作者变现
```

### 4.2 为什么不把你当竞品

| 公司 | 他们的核心 | 你的核心 | 冲突？ |
|------|-----------|---------|--------|
| 谜圈 | 组局+评分（流量撮合） | 店务+粉丝经济（运营） | ❌ 互补 |
| 千岛 | 潮玩交易（剧本杀是频道之一） | 沉浸式娱乐创作者经济 | ❌ 不同赛道 |
| 米花星球 | UGC 引擎（让人做本） | 让人管店、管人、管粉丝 | ❌ 不同方向 |
| 剧帮主 | 组局+店务（功能全但无亮点） | 粉丝经济驱动 | ⚠️ 部分重叠但你更深 |
| 纳客/云上铺 | 通用 SaaS 套壳（不专业） | 行业深度定制 | ✅ 你的优势 |

### 4.3 壁垒不是功能，是飞轮

```
卡司入驻 → 粉丝关注 → 卡司赚到钱 → 更多卡司入驻
                      ↕
            店家为卡司而来 → 店家付费 → 店家提供更多场次
                      ↕
            玩家为卡司而来 → 玩家付费打榜
```

这个飞轮一旦转动，后来者很难复制。不是功能难抄，是**双边网络效应**难抄。

## 五、对标参考

### 5.1 最直接的对标：秀动（太合音乐）

秀动做的事：独立音乐人 × Livehouse × 粉丝

| 维度 | 秀动 | 剧掌柜 |
|------|------|--------|
| 创作者 | 独立音乐人 | 沉浸式卡司 |
| 场地 | Livehouse | 演绎剧场/密室等 |
| 粉丝 | 找演出、买票 | 追卡司、预约、打榜 |
| 商业模式 | 票务抽成 | SaaS + 粉丝经济抽成 |
| 粉丝经济 | ❌ 没做 | ✅ 打榜/委托/粉丝团 |

### 5.2 粉丝经济借鉴：B 站 / 抖音

| 机制 | 借鉴来源 | 你的版本 |
|------|---------|---------|
| 打赏 | 抖音直播礼物 | 卡司打榜 |
| 粉丝团 | B 站大航海 | 卡司月卡订阅 |
| 创作者变现 | 星图广告 | 可接委托 |
| 排名 | 抖音小时榜 | 卡司热度榜 |

### 5.3 增长模式借鉴：有赞

| 阶段 | 有赞做法 | 你可以学的 |
|------|---------|-----------|
| 冷启动 | 免费开店工具吸引商家 | 免费卡司主页吸引卡司 |
| 增长 | 商家多了→消费者多→更多商家 | 卡司多了→粉丝多→店家入驻 |
| 变现 | 基础免费+增值收费 | 基础排班免费/低价+粉丝经济抽成 |

## 六、商业模式

### 6.1 收入模型

| 收入来源 | 付费方 | 预期占比 |
|---------|--------|---------|
| 店家 SaaS 订阅 | 店家 | 40% |
| 打榜抽成（15%） | 玩家 → 卡司 | 30% |
| 委托抽成（15%） | 玩家 → 卡司 | 20% |
| 粉丝团抽成（15%） | 玩家月费 | 10% |

### 6.2 单店经济模型（SaaS 部分）

| 项目 | 数值 |
|------|------|
| 基础版 | 免费 |
| 专业版（占比 50%） | ¥599/月 |
| 旗舰版（占比 20%） | ¥1,999/月 |
| 加权月 ARPU | ~¥700 |
| 年 ARPU | ~¥8,400 |

### 6.3 粉丝经济抽成模型

假设一个演绎剧场有 500 个活跃玩家关注其卡司：

| 项目 | 保守 | 中性 | 乐观 |
|------|------|------|------|
| 打榜渗透率 | 5% | 10% | 20% |
| 月人均打榜金额 | ¥50 | ¥100 | ¥200 |
| 场均打榜收入 | ¥1,250 | ¥5,000 | ¥20,000 |
| 平台月抽成（15%） | ¥188 | ¥750 | ¥3,000 |

单店粉丝经济抽成就可能超过 SaaS 订阅收入。

### 6.4 里程碑

| 阶段 | 时间 | 目标 | 关键指标 |
|------|------|------|---------|
| 冷启动 | 0-3 月 | 50 个卡司入驻 + 5 家店试用 | 卡司周活 > 60% |
| 粉丝经济启动 | 3-6 月 | 200 个卡司 + 20 家付费店 | 首单打榜出现 |
| 飞轮转动 | 6-12 月 | 1000 个卡司 + 100 家付费店 | MRR > 7 万 |
| 规模化 | 12-24 月 | 5000 卡司 + 500 家店 | MRR > 35 万，盈亏平衡 |

## 七、风险与应对

| 风险 | 级别 | 应对 |
|------|------|------|
| 卡司入驻后没有粉丝互动 | 🟡 中 | 先打样 5-10 个有粉丝基础的卡司，验证闭环再规模推广 |
| 店家不愿意付费 | 🟡 中 | 基础排班免费，店家为高级功能和卡司数据付费 |
| 沉浸式娱乐赛道太小 | 🟢 低 | 赛道在快速扩张；且你做的是创作者经济平台，不限于一种业态 |
| 大厂入场 | 🟢 低 | 市场太小巨头看不上；等他们看上的时候你已经有了网络效应 |
| 打榜模式被监管 | 🟡 中 | 参考直播打赏监管框架，提前做好合规设计 |

## 八、现在你该做什么

1. **这周**：压出 10 页 Pitch Deck
2. **两周内**：找到 5 个有粉丝基础的卡司（而不是找店家），给他们看卡司主页设计稿，问他们愿不愿意用
3. **一个月内**：做出卡司个人主页 MVP（免费开放入驻），用卡司吸引店家
4. **三个月内**：跑通第一个「卡司入驻 → 粉丝关注 → 打榜/委托」的闭环

**先验证「卡司愿意用」，再验证「粉丝愿意付钱」，最后验证「店家愿意付费」。顺序不能错。**
