# 剧掌柜 · 泛二次元创作者经济平台

> **版本**：BP v3.0（终版）  
> **日期**：2026-05-04  
> **一句话**：让每个把纸片人带到现实的人，都能经营粉丝、接单变现。

---

## 一、我们发现了什么

### 1.1 一个正在爆发但没人服务的群体

中国有超过 **5 亿泛二次元用户**。这群人愿意为「纸片人走进现实」付费。

已经跑通的业态：

```
沉浸式剧场（剧本杀/演绎剧场）
  → 卡司/DM 是明星 → 幽州剧场打榜最高 24 万/人

Cosplay 委托
  → 小红书 16 亿浏览量 → 单日委托 1000+ 元
  → 陪伴经济市场 500 亿元（2025）

Cosplay 产业链
  → 全球 Cos 服装市场 47.5 亿美元
  → 妆娘、毛娘、摄影师配套职业化

剧本杀展会
  → 明星 DM 特邀开本 → 粉丝追着跑
```

**这些人的共同点：他们都是「把纸片人带到现实」的创作者。**

但他们现在用的工具极其分散：

| 需求 | 现在用的 | 问题 |
|------|---------|------|
| 展示自己 | 小红书/微博/B站 | 算法不推就没人看 |
| 接委托 | 微信私聊 | 没有标准价格、没有担保 |
| 排档期 | 手记/微信群 | 跨店跨展就乱 |
| 收钱 | 个人收款码 | 没有平台背书 |
| 粉丝管理 | 各平台分散 | 无法沉淀 |

**没有一个平台是专门为这群人做的。**

### 1.2 竞品分析：为什么他们不是竞品

| 公司 | 他们在做什么 | 为什么不是竞品 |
|------|------------|--------------|
| **巨量星图** | 品牌方 → 达人 → 拍视频 → 线上传播 | 你做的是粉丝 → 创作者 → 线下体验 |
| **谜圈** | 组局+评分（流量撮合） | 重平台轻工具 |
| **千岛** | 潮玩交易（剧本杀只是频道之一） | 不做创作者经济 |
| **米花星球** | UGC 引擎（让人做本） | 不做线下交易 |
| **剧帮主/纳客** | 店务管理工具 | 不做粉丝生态 |

**结论：没有直接竞品。这个赛道是空的。**

## 二、产品

### 2.1 一句话

> **剧掌柜 = 泛二次元创作者的个人官网 + 粉丝经济 + 交易平台。**

### 2.2 先服务谁（MVP）

**第一期只服务创作者（卡司/Coser），不做店家端。**

```
MVP 功能：
├── 个人主页（头像、简介、作品集、社交链接）
├── 可接委托（明码标价、服务类型）
├── 档期日历（哪天有空）
└── 「约我」引导（跳转微信，不在平台内聊天交易）
```

**为什么从创作者切入：**
- 一个人就能注册，无需审批
- 有展示需求，有变现需求
- 创作者多了，店家自然来找

### 2.3 完整蓝图

```
创作者端（卡司/Coser/摄影师/妆造师）—— 免费入驻
├── 个人主页（作品集、风格标签、评分、粉丝数）
├── 可接委托（明码标价，平台担保交易）
├── 档期管理（跨店/跨展）
├── 粉丝数据（粉丝数、打榜排名、热度曲线）
├── 粉丝团订阅制
├── 社交链接合集（抖音/小红书/B站/微博）
└── 成长体系（等级、成就、认证）

玩家/粉丝端
├── 发现创作者（按风格/城市/热度）
├── 创作者主页（看作品、看档期、约委托）
├── 关注/订阅
├── 打榜/送礼
├── 粉丝团订阅
├── 历史参与记录
└── 评价体系

店家/场地端（第二阶段）
├── 场地展示（声光电、装修、道具模组、剧本库）
├── 驻场创作者列表
├── 排期管理
└── 数据看板
```

## 三、商业模式

### 3.1 收入来源

| 收入项 | 说明 | 预期占比 |
|-------|------|---------|
| 创作者增值服务 | 高级主页、数据分析、推广位 | 20% |
| 委托交易抽成（5-10%） | 平台担保交易后抽成 | 40% |
| 打榜/礼物抽成（15%） | 粉丝给创作者打赏 | 25% |
| 店家 SaaS 订阅 | 第二阶段 | 15% |

### 3.2 MVP 阶段（0 收入）

卡司/Coser 入驻完全免费。
委托引导到微信，零交易、零抽成、零合规风险。

**先验证「有人愿意用」，再验证「有人愿意付钱」。**

### 3.3 里程碑

| 阶段 | 时间 | 核心目标 |
|------|------|---------|
| MVP | 0-2 月 | 50 个创作者入驻，完成个人主页 |
| 验证 | 2-4 月 | 200 个创作者，出现第一笔通过平台引导的委托 |
| 交易 | 4-6 月 | 上线平台内担保交易，开始抽成 |
| 飞轮 | 6-12 月 | 1000+ 创作者 + 100+ 店家入驻 |

## 四、合规策略

### 4.1 MVP 阶段 —— 不需要任何牌照

只做「个人主页展示 + 引流到微信」，不涉及：

- ❌ 平台内支付 → 不需要支付牌照
- ❌ 平台内聊天 → 不需要社交资质
- ❌ 平台内交易 → 不需要电商许可证
- ❌ UGC 海量内容 → 人工审核前 100 个创作者，零成本

### 4.2 交易阶段 —— 找持牌合作

当需要做平台内交易时，不自己申请支付牌照（几乎不可能），而是：
- 对接微信支付/支付宝的服务商模式
- 或与持牌机构合作（如 Ping++、连连支付）

### 4.3 内容审核

- MVP 阶段：创始人自己审
- 增长阶段：外包审核团队
- 规模阶段：AI 审核 + 人工复审
